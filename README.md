# flow
Change the colour of a LIFX bulb based on playback from Spotify

usage: flow.py [-h] [-s] [-b]

optional arguments:
  -h, --help     show this help message and exit
  -s, --strobe   Strobe to the BPM of a song (changes based on tempo
                 changes)
  -b, --breathe  With breathe effect
